It's better to hang yourself than make a config for hyprland


## Installation
#### step 1
~~~bash
sudo pacman -S rofi waybar swaync swww nautilus kitty fish bc cava ttf-firacode-nerd noto-fonts ttf-jetbrains-mono-nerd ttf-font-awesome noto-fonts-emoji
~~~
#### step 2
~~~bash
git clone https://github.com/NyxAiko7/config-hypr.git
cd config-hypr
~~~
#### step 3
~~~bash
cp -r hypr ~/.config
cp -r waybar ~/.config
cp -r kitty ~/.config
cp -r swaync ~/.config
~~~

## Warning
don't forget to set your screen resolution
