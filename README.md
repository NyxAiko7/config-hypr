## Installation
#### step 1
~~~bash
sudo pacman -S viu swayosd fuzzel waybar swaync swww nemo alacritty  fish bc cava ttf-firacode-nerd noto-fonts ttf-jetbrains-mono-nerd ttf-font-awesome noto-fonts-emoji python-pywal
yay -s wpgtk
~~~
#### step 2
~~~bash
git clone https://github.com/NyxAiko7/config-hypr.git
cd config-hypr
~~~
### step 2.5

~~~bash
wal -i Let's go to the wallpaper
~~~

#### step 3
~~~bash
sudo cp -r wal ~/.config
sudo cp -r hypr ~/.config
sudo cp -r waybar ~/.config
sudo cp -r kitty ~/.config
sudo cp -r swaync ~/.config
sudo cp -r fuzzel ~/.config
sudo cp -r ~/.local/share/applications/img.desktop
sudo cp -r /usr/local/bin/img
~~~
